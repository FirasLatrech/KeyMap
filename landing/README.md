# KeyMap Fix — Landing page

Single-scroll Next.js + Tailwind landing for the KeyMap Fix Raycast extension.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Stack

- Next.js 15 (App Router) + React 19
- Tailwind CSS 3
- Geist Sans + Geist Mono (Latin), IBM Plex Sans Arabic (RTL)

## Design tokens

See `tailwind.config.ts`. Palette: `#0A0A0A` bg, `#141414` surface, `#7C5CFF` accent.
