First release of **KeyMap Fix** for macOS.

## Install

1. Download `KeyMap-1.0.0.dmg` below.
2. Open it, drag **KeyMap.app** into **Applications**.
3. First launch: right-click → **Open** → **Open Anyway** (unsigned).
4. Grant Accessibility permission when prompted.
5. Select any text, press **⌥⌘K** — done.

## What's in this release

- Native SwiftUI menu-bar app (Apple Silicon + Intel universal), macOS 13+.
- Global hotkey **⌥⌘K** to convert your selection in place.
- Layouts: **AR ↔ EN** (macOS Arabic), **FR ↔ EN** (AZERTY).
- 100% local. No network. No telemetry.

## Notes
