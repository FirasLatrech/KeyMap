# KeyMap Fix

A macOS Raycast extension that converts mis-typed text between Arabic, English, and French (AZERTY) keyboard layouts in one hotkey.

## Layout

- [`raycast-extension/`](./raycast-extension) — the Raycast extension (TypeScript + React).
- [`landing/`](./landing) — the marketing site (Next.js 15 + Tailwind).
- [`DESIGN.md`](./DESIGN.md) — design brief for v0 / Lovable / Figma.

## Quick start

```bash
# Landing page
cd landing && npm install && npm run dev

# Raycast extension
cd raycast-extension && npm install && npm run dev
```
